A mental wellness support platform designed for students. Provides a **safe, private, and calming space** to express emotions and receive immediate support.

---

## Problem
Students face **stress, anxiety, and academic pressure**, but often hesitate to seek help. Professional support is not always available.  
**Need:** Accessible, non-judgmental, always-available emotional support.

---

## Solution
An AI-powered assistant that:  
- Detects emotions via chat  
- Provides empathetic messages  
- Offers wellness tools like affirmations, meditation, and breathing exercises  
- Features a calming, distraction-free UI  

---

## Key Features
- Emotion-aware chat support  
- Daily positive affirmations  
- Guided meditation  
- Breathing exercises  
- Soothing UI  

---

## How It Works
- **NLP-based emotion detection**  
- **Dynamic responses** based on emotional patterns  
- Frontend: HTML, CSS, JavaScript  
- AI logic: Rule-based (can integrate LLMs for advanced support)  

---

## Future Scope
- LLM-based AI for personalized responses  
- Mood tracking dashboards  
- Personalized wellness plans  
- Emergency resource redirection  
- Integration with college counseling systems  

---

## Note
This project **does not replace therapists** or medical advice. It provides **first-level emotional support** to encourage self-care.
